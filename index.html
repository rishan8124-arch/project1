Web Design Process

The image illustrates the key components involved in the web design process. These components include:

1. Usability: Ensuring the website is user-friendly and easy to navigate.
2. Layout: The arrangement of visual elements on the website.
3. Internet: The online environment where the website will be accessed.
4. CSS (Cascading Style Sheets): A style sheet language used for describing presentation.
5. CMS (Content Management System): Software used to create and manage digital content.
6. Content: Information and media presented on the website.
7. Media: Images, videos, audio, and other formats.
8. Homepage: The main starting point of navigation.

These elements work together to create a functional and aesthetically pleasing website.

Interaction Design Process

The image illustrates the key principles of the Interaction Design Process:

1. Learnable: Easy for users to understand.
2. Consistent: Interface and design remain uniform.
3. Predictable: Users can guess what happens next.
4. Perceivable: All UI elements are clear and noticeable.
5. Feedback: System gives responses to user actions.

These principles work together to improve user interaction and experience.
